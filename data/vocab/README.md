# Словари (vocab sets)

Каждая подпапка здесь — отдельный набор карточек.

## Как добавить новый набор

1. Создай папку с латинским slug-именем (только латиница, цифры и дефисы):
   `data/vocab/family-members/`
2. Внутри положи файл `index.json`:

```json
{
  "title": "Семья",
  "title_id": "Keluarga",
  "description": "Слова, обозначающие членов семьи",
  "description_id": "Kata yang menyatakan anggota keluarga",
  "words": [
    {
      "id": "ayah",
      "en": "father",
      "ru": "отец",
      "examples": [
        { "id": "Ayah saya seorang guru.", "ru": "Мой отец — учитель." },
        { "id": "Saya cinta ayah saya.", "ru": "Я люблю моего отца." }
      ]
    },
    { "id": "ibu", "en": "mother", "ru": "мать" },
    { "id": "anak", "en": "child", "ru": "ребёнок", "note": "сын или дочь, рода нет" }
  ]
}
```

3. Закоммить и запушь — Vercel передеплоит автоматически. Набор появится на `/vocab`.

## Поля

| Поле | Обязательное | Описание |
|------|--------------|----------|
| `title` | да | Заголовок набора (на русском) |
| `title_id` | нет | Заголовок на индонезийском (для ID-режима интерфейса) |
| `description` | нет | Подзаголовок на русском |
| `description_id` | нет | Подзаголовок на индонезийском |
| `words[].id` | да | Слово или фраза на индонезийском (будет озвучено на id-ID) |
| `words[].en` | нет | Английский перевод (будет озвучен на en-US) |
| `words[].ru` | да | Перевод на русский |
| `words[].note` | нет | Пометка (напр., «формально», «разговорное») |
| `words[].examples` | нет | До 3 примеров `{id, ru}` — показываются под словом на карточке |

Регистр сохраняется. Можно ставить целые фразы — `id` будет произнесён целиком.
